# Bruteforce
Bruteforce em sites

Copie o diretório `Login` para a pasta `/sdcard/htdocs`, inicie o servidor web e acesse o endereço `localhost:8080/Login` Ou `https://localhost:8443/Login` caso o certificado tiver ativo.
